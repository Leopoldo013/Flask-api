Univesp

Documentação api-prodesp

Api para acesso de dados bolsa trabalho e via rápida

Versão: 1.0.0

[ Base URL: https://assets.univesp.br/api-prodesp ]

Descrição:

Para acessar os dados, primeiro temos que efetuar o post com o usuario e senha em formato json para obter o token JWT.

Exemplo:

          
            {
              'username':'username'
              'password':'password'
            }

Resposta

          
            {
              "token": "token"
            }

Descrição:

Necessario para passar o token via header para efetuar o get.

Essa rota retorna todos os dados de todos os cursos.

Exemplo:

          
            {
              JWT:token
            }

Resposta

          
               {
                    "FIRST_NAME": "string",
                    "LAST_NAME": "string",
                    "FIRST_ACCESSED_TIME": "string",
                    "BIRTH_DATE": "string",
                    "EMAIL": "string",
                    "NAME_COURSE": "string",
                    "DURATION_SUM": "number",
                    "ID_COURSE": "string",
                    "COURSE_NUMBER": "string",
                    "TIME": "string",
                    "PERSON_ID": number,
                    "INTERACTION_CNT": number
                }

Descrição:
